# Mine Safety System
## 🛡️ Project Overview
A web-based mine safety system with worker login and supervisor dashboard functionality.
## Features
- 🔐 Worker Login System
- 📋 Daily Safety Checklist
- 🚨 Emergency SOS with GPS Location
- 👨‍💼 Supervisor Dashboard
- 📊 Real-time Safety Monitoring
## Technologies Used
- HTML5
- CSS3
- JavaScript
- Firebase (Real-time Database)
## How to Use
1. Open `index.html` in a web browser
2. Login as worker or supervisor
3. Use the safety monitoring features
